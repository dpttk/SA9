# Req-s

Docker Comunity Edition

# Run Services

1) clone repo
2) add your email credits to env file (services/env and pipes-filters/env)
3) deploy docker via `docker compose up -d` in services or pipes directory

# Run Test

For services

```
cd services
./check.sh
```

for pipes&filters

```
cd pipes-filters
./check.sh
```
